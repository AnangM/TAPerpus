Bantuan Login

PERHATIAN user TIDAK DAPAT membuat akun baru, hanya Super User yang dapat membuat akun baru untuk Login.


Untuk Login user harus memasukkan username dan password yang bersangkutan. User tidak dapat melakukan bypass dengan "=''or'. Apabila lupa password maka harus menghubungi Super User yang bersangkutan. Setelah user mengisi username dan password maka kilik tombol login. Apabila benar maka akan menuju ke halaman utama, apabila gagal maka harus mengulang login.
